# The Index

## משתנים (Variables)

> לתפוס תא קטן בזיכרון של המחשב ולתת לו שם, כדי לשמור בו ערך שנרצה להשתמש בו מאוחר יותר.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)

* **אפשרות א' (כוס מדידה):** אתה מוציא כוס מדידה מהארון לאפות עוגה. לא אכפת לך אם יש בה קמח או סוכר – אתה פשוט יודע שזו ה"כוס" שתלך לתערובת. השתמשת? סיימת? היא חוזרת לארון.
* **אפשרות ב' (תווית על קופסה):** דמיינו שאתם מסדרים ארגז מעבר דירה. אתם לא צריכים לזכור שבקופסה מסוימת יש כלי מטבח – אתם סתם כותבים "מטבח" על התווית. משתנה הוא אותה תווית בזיכרון של המחשב.

### 💻 קוד תמציתי (Python)

```python
measuring_cup = "Flour"  # Created a variable and assigned a value
print(measuring_cup)     # Flour

measuring_cup = "Sugar"  # Reassigned – same variable, new content
print(measuring_cup)     # Sugar

CAKE_NAME = "Birthday"   # Convention: ALL_CAPS = treat this as a constant, don't change it
print(CAKE_NAME)         # Birthday
```

> **לשים לב:** בפייתון אין `const` כמו ב-JavaScript – אבל כשרוצים לסמן שמשתנה הוא קבוע (לא אמור להשתנות), כותבים אותו ב-UPPERCASE. זו מוסכמה בין מפתחים, ולא חוק של השפה.

### 🛠️ יישום בפרויקט

בבניית **מחשבון**: המשתנה הוא המקום שבו המחשבון שומר את המספר הראשון שהקלדת, בזמן שאתה מקליד את המספר השני. בלי המשתנה, הוא היה "שוכח" את המספר הראשון ברגע שעברת לשדה הבא.
