# The Index - ויקי המושגים של Live Code

ויקי טכנולוגי שיתופי ופתוח לקהילת המפתחים והג'וניורים – פשוט, מזוקק ובגובה העיניים.

---

## 🌳 מבנה 3 השכבות (Architecture)

* **שכבה 1: תיקיות לפי נושאים 📁** – החלוקה הראשית של הפרויקט לתיקיות על לפי תחומים (למשל: `04. DevOps & Cloud`).
* **שכבה 2: קובץ הויקי / ערך נושא (רמת סניור) 📄** – הקובץ המרכזי שמחבר את התמונה הגדולה, נותן את ההקשר המקצועי ומאגד תחתיו את ה"ברגים".
* **שכבה 3: הברגים ⚙️** – המונחים האטומריים, הפקודות המעשיות וכלי הקצה (כמו `grep`, `cat` או `strings`) שמרכיבים את הנושא בפועל.
