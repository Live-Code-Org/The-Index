# 📝 The Index
המטרה של הקובץ הזה היא לרכז את  כל "הברגים" (המושגים, הפקודות והמונחים) שלמדנו ונלמד או סתם יוצא לנו להכיר, במהלך הלימודים.

### 🛠️ איך מוסיפים מושג חדש?
1. מצאו את הקטגוריה המתאימה ברשימה למטה.
2. הוסיפו את המושג בשורה חדשה. ודאו שהשורה מתחילה בדיוק בסימן הבא: מקף, אז רווח, ואז תיבה ריקה `[ ]` (כמו בשורות המוכנות למטה).
3. אם מדובר בתת-נושא (כמו פקודת לינוקס), תנו **שני רווחים (הזחה)** בתחילת השורה לפני המקף.
4. אם יש לכם כוח, כתבו לידו ב-2 מילים מה זה (למשל: `cat` - הצגת תוכן קובץ).
