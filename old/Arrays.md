# The Index


## מערכים (Arrays)

>רשימה של נתונים שיושבים תחת שם אחד, מסודרים עם מספר סידורי (אינדקס).



### אנלוגיה מהחיים
**מגירת סכו"ם.** במקום שיהיה לך משתנה לכל מזלג (fork1, fork2), יש לך מגירה אחת שנקראת "סכו"ם ". המיקום של כל כלי הוא קבוע – המזלגות תמיד בתא הראשון (אינדקס 0), הסכינים בשני, וכן הלאה. אם אתה רוצה להוסיף כף, אתה פשוט דוחף אותה למגירה  
 (כף נוספת - למשתנה הכפות, כלי חדש - כמשתנה חדש במערך ) .


### קוד תמציתי (JS)

```js
const drawer = ["Fork", "Knife", "Spoon"];

console.log(drawer[0]); // "Fork"
drawer.push("Teaspoon"); // הוספנו כלי למגירה
```



## יישום בפרויקט
בבניית **רשימת משימות (Todo List):** המערך הוא הזיכרון של הרשימה. כל משימה חדשה שהמשתמש מקליד נכנסת כ"איבר" נוסף למערך. ככה אנחנו יכולים לעבור על כל המערך ולהדפיס את כל המשימות על המסך בפעולה אחת.
