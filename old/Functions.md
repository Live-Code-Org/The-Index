## פונקציות (Functions)


>מכונה שמוגדרת מראש. נותנים לה חומרים (קלט), היא עושה עליהם סיבוב, ומוציאה תוצר (פלט).



### אנלוגיה מהחיים

**בלנדר**. אתה לא בונה בלנדר מחדש בכל פעם שבא לך שייק. הבלנדר הוא מנגנון קבוע. אתה זורק פנימה פירות (נתונים), לוחץ על כפתור (קריאה לפונקציה), ומקבל שייק מוכן (תוצאה).


### קוד תמציתי (JS)

```js
function makeShake(fruit) {      // function = הגדרת פונקציה , fruit = פרמטר (הקלט)
  return fruit + " Shake ";     // return = מה הפונקציה מחזירה
}

const myDrink = makeShake("Banana");     // קריאה לפונקציה עם ארגומנט    
console.log(myDrink);                   // "Banana Shake "
```
