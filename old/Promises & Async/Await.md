## הבטחות (Promises & Async/Await)

>הסכם מול המחשב: "שלחתי פעולה שתיקח זמן. אל תחכה לה ותעצור הכל – תמשיך לעבוד, וכשהיא תחזור, תעדכן אותי".



### אנלוגיה מהחיים
**מילוי בקבוק מים במיניבר.**  
 הדרך ה"חסומה" (Sync): אתה לוחץ על הכפתור ובוהה בבקבוק עד שהוא מלא. כל הבית מחכה לך, שום דבר אחר - לא קורה.

הדרך ה"מבטיחה" (Async): אתה לוחץ על הכפתור (שולח את ה-Promise). אתה הולך לסדר את החדר (ממשיך להריץ קוד אחר). המכשיר "מבטיח" לך שהוא יסיים. כשתשמע את הצפצוף שהבקבוק מלא – תחזור לקחת אותו.

### קוד תמציתי (JS)

```js
async function fillWaterAndClean() {
  // "לוחצים" על המיניבר ולא מחכים לו (Promise)
  const waterPromise = fillBottle(); 

  // בינתיים מסדרים את החדר (קוד שלא חסום)
  cleanRoom(); 

  // עכשיו "מחכים" (await) לראות שהבקבוק אכן התמלא
  const bottle = await waterPromise; 
  console.log("Room is clean and bottle is " + bottle);
}
```
## יישום בפרוייקט


בבניית **אפליקציית חדשות:** אנחנו לא רוצים שהמסך יהיה לבן וקפוא בזמן שהאפליקציה מורידה את הכתבות מהאינטרנט. אנחנו מציגים למשתמש את מבנה האתר (סידור החדר) ורק כשהכתבות חוזרות מהשרת (הבקבוק התמלא) – אנחנו "מזריקים" אותן למסך.
