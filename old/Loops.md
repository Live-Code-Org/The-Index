## לולאות (Loops: For / While)

>מנגנון שמריץ קטע קוד שוב ושוב, כדי שלא נצטרך לכתוב את אותה שורה מאה פעמים (או אינסוף םעמים...)



### אנלוגיה מהחיים
- לולאת For : כמו מכונת דפוס. אתה אומר לה: "תוציאי 1,000 עותקים". המספר ידוע מראש, המכונה סופרת, ובסוף היא עוצרת מעצמה.
- לולאת While : כמו חיישן אור במרפסת. הוא לא יודע כמה זמן הוא יעבוד. הוא "רץ" כל עוד חשוך. ברגע שהשמש זורחת (התנאי משתנה) – הוא מפסיק.


### קוד תמציתי (JS)

```js
// For - ידוע מראש כמה חזרות
for (let i = 1; i <= 1000; i++) {
  printNewspaper(i); 
}

// While - תלוי במצב משתנה
let personInBalcony = true;
while (personInBalcony) {
  keepLightOn();
  personInBalcony = checkSensor(); // בודק אם המצב השתנה
}
```
#### כלים נוספים

- הפקודה `break` (שבירה מיידית): עוצר את הלולאה לגמרי ויוצא החוצה מיד.  

- הפקודה `continue` (דלג על זה): עוצר את הסיבוב הנוכחי וקופץ ישר לתחילת הסיבוב הבא.  

+ לולאה בתוך לולאה (Nested Loops): כמו ניקיון בניין, עוברים קומה-קומה, ובכל קומה עוברים חדר-חדר.

**לדוגמה -**

```js
for (let floor = 1; floor <= 10; floor++) {    // לולאה חיצונית: עוברת על הקומות
  for (let room = 1; room <= 5; room++) {    // לולאה פנימית: עוברת על החדרים בכל קומה
    if (isRoomLocked) continue;    // החדר נעול? דלג ישר לחדר הבא
    if (isEmergency) break;    // מקרה חירום? עצור את כל העבודה וצא מהבניין
    clean(floor, room);    // הפעולה שמתבצעת בכל חדר
  }
}
```

## יישום בפרויקט

-  לולאת `For` : לעבור על רשימת מוצרים בסל הקניות ולהדפיס את השם של כל אחד מהם.

+ לולאת `While` : בבניית צ'אט: הלולאה תמשיך לרוץ ולחכות להודעות חדשות כל עוד המשתמש לא לחץ על "ניתוק".
