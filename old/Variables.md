# The Index


## משתנים (Variables)


>לתפוס תא קטן בזיכרון של המחשב כדי לשמור בו משהו לרגע.



### אנלוגיה מהחיים

**כוס מדידה.** אתה מוציא אותה מהארון כדי לאפות עוגה. כרגע לא אכפת לך אם יש בה קמח או סוכר – אתה פשוט יודע שזו ה"כוס" שתלך לתערובת. השתמשת? סיימת? היא חוזרת לארון.


### קוד תמציתי (JS)

```js
let measuringCup = "Flour"; // הוצאנו כוס, שמנו קמח
measuringCup = "Sugar";     // שפכנו, שמנו סוכר (אותה כוס)
const cakeName = "Birthday"; // שם העוגה - לא משתנה לעולם
```

> **לשים לב:** אנו נגדיר משתנים באופן שונה בהתאם למטרת המשתנה:

- נגדיר `const`: עבור ערכים קבועים שלא נרצה שיוכלו לעבור עריכה חופשית. (כמו שם, כתובת )
- נגדיר  `let`:  עבור ערכים שחייבים להשתנות באופן תדיר (כמו ערכים בלולאות, או ערך שמתעדכן).


## יישום בפרויקט


בבניית **מחשבון** המשתנה הוא המקום שבו המחשבון שומר את המספר הראשון שהקלדת, בזמן שאתה עסוק בלהקליד את המספר השני. בלי ה"כוס" הזו, הוא היה שוכח מה רצית לפני שנייה.
