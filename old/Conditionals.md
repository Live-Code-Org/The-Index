## תנאים (Conditionals - If/Else)


>צומת בקוד. המחשב בודק אם משהו -או חלקו - נכון ׁ(-תנאי), ולפי זה מחליט באיזה נתיב להמשיך לרוץ.


### אנלוגיה מהחיים

**הכניסה למועדון.** המאבטח (הקוד) בודק שני דברים: "האם יש לך כרטיס?" וגם "האם אתה מעל גיל 18?".

+ אם שניהם נכונים `AND` – אתה נכנס.

- אם יש לך כרטיס VIP או שאתה חבר של בעל המקום `OR` – אתה נכנס גם בלי התור.

בכל מקרה אחר (ELSE) – אתה נשאר בחוץ.

### קוד תמציתי (JS)

```js
const age = 20;
const hasTicket = true;

if (age >= 18 && hasTicket) {    // בדיקה אם התקיימו 2 תנאים
    console.log("Welcome in!");   
}else if (isVIP || isFriend) {   //אם לא התקיימו, בדיקה אם התקיים לפחות תנאי אחד אחר 
    console.log("Welcome in!"); 
}else{                           // אם אף תנאי לא התקיים
  console.log("Go home... ");
}
```

> **שים לב!** לפעמים נוכל לצמצם בדיקת תנאי לשורה אחת בלבד.

לדוגמה - 
```js
const access = (age >= 18) ? "Allowed" : "Blocked"; // Ternary: השמת ערך משתנה לפי אמת או שקר


if (isVIP || isFriend) enterDoor(); // תנאי אמת בלבד: ביצוע פעולה ישירה ללא סוגריים מסולסלים
```
