# The Index


## טווח הכרה (Scope)

>הגבולות שבהם המחשב "מכיר" משתנה מסוים. אם יצאת מהגבול – המשתנה לא קיים..



### אנלוגיה מהחיים
כינוי משפחתי מול שם רשמי. הכינוי הילדותי שלך (נניח "צ'וצ'ו") קיים רק בתוך הבית. אם מישהו יצעק "צ'וצ'ו" ברחוב, אף אחד לא יסתובב. לעומת זאת, השם שלך בתעודת הזהות הוא גלובלי – כולם מכירים אותו, גם בבית וגם בבנק.

### קוד תמציתי (JS)

```js
const globalName = "Michael"; // שם רשמי - כולם רואים

function home() {
  const nickName = "ChooChoo"; // כינוי - רק בתוך הפונקציה
  console.log(globalName);     // עובד! בבית מכירים את השם הרשמי
  console.log(nickName);       // עובד!
}

home();
console.log(nickName); // שגיאה! (ReferenceError) העולם לא מכיר את צ'וצ'ו
```




## יישום בפרויקט
בבניית **מערכת תשלומים:** שם המשתמש יהיה לרוב גלובלי (כדי שיופיע בכל דף באתר), אבל משתנה כמו temporaryDiscount (הנחה זמנית) יוגדר רק בתוך הפונקציה של "חישוב המחיר בקופה". אנחנו לא רוצים שחלקים אחרים באתר יוכלו לגשת אליו או לשנות אותו בטעות.
