## כתובת IP (IP Address)

> מספר זיהוי ייחודי שניתן לכל מכשיר המחובר לרשת, המאפשר לנתב אליו מידע בדיוק – כמו כתובת דואר במרחב הדיגיטלי.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)

* **אפשרות א' (כתובת בית לדוור):** כדי שמכתב יגיע אליכם, הדוור צריך מדינה, עיר, רחוב ומספר בית. כתובת IP היא בדיוק זה – כתובת מדויקת של מכשיר ברשת, כדי שחבילות מידע ידעו לאן להגיע.
* **אפשרות ב' (מספר טלפון):** לכל אדם יש מספר ייחודי שדרכו מגיעים אליו. כך גם לכל מחשב, טלפון או שרת – יש מספר IP ייחודי שדרכו ניתן "לחייג" אליו ולשלוח מידע.

### 💻 קוד תמציתי (Python)

```python
import socket

# Finding the local IP address of this machine
hostname   = socket.gethostname()
local_ip   = socket.gethostbyname(hostname)

print(f"Machine name: {hostname}")  # e.g. my-laptop
print(f"Local IP:     {local_ip}")  # e.g. 192.168.1.15

# 127.0.0.1 is a special address: it always means "this machine" (localhost)
loopback = "127.0.0.1"
print(f"Loopback: {loopback}")
```

> **לשים לב:** כשאתם מריצים שרת מקומי ורואים `http://127.0.0.1:3000` – זו כתובת ה-IP של המחשב שלכם *עצמו*. כלומר, הדפדפן שלכם מדבר עם השרת שלכם על אותו מחשב. נקרא גם `localhost`.

### 🛠️ יישום בפרויקט

בזמן פיתוח, השרת המקומי שלנו מאזין על `127.0.0.1`. כשנעלה לאוויר (Deploy), הוא יקבל כתובת IP ציבורית מהספק (AWS, Heroku וכו'). שם בדיוק נכניס את כתובת ה-IP בהגדרות ה-DNS כדי שהאתר שלנו יהיה נגיש לכולם.
