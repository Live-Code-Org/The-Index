## DNS (Domain Name System)

> מערכת שמות המתחם – שירות שמתרגם שמות אתרים קריאים לבני אדם (כמו `google.com`) לכתובות IP שהמחשב מבין.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)

* **אפשרות א' (אנשי קשר בנייד):** אף אחד לא זוכר את מספר הטלפון המלא של חבריו. אנחנו שומרים "אמא" ו"יוסי", והטלפון מתרגם את השם למספר ומחייג. DNS עושה בדיוק אותו דבר: אתם כותבים `youtube.com`, הוא מתרגם לכתובת IP ומחבר אתכם.
* **אפשרות ב' (שירות מודיעין 144):** "אני רוצה להגיע לשגרירות ישראל בברלין." המוקדן מחפש, מוצא את הכתובת המדויקת ומחבר אתכם. אתם לא צריכים לדעת את הכתובת מראש – שירות ה-DNS מחפש עבורכם.

### 💻 קוד תמציתי (Python)

```python
import socket

# DNS lookup: translating a domain name to an IP address
domain = "livecode.org"
ip     = socket.gethostbyname(domain)

print(f"Domain: {domain}")  # livecode.org
print(f"IP:     {ip}")      # e.g. 104.21.83.123

# This translation happens automatically in your browser
# every time you visit a website
```

> **לשים לב:** כל פעם שאתם גולשים לאתר, מתרחשת שאילתת DNS שקטה ברקע לפני שהדף נטען. לפעמים כשאתר "לא נטען" זה דווקא בעיית DNS ולא בעיית השרת עצמו – ניתן לבדוק עם `nslookup` בטרמינל.

### 🛠️ יישום בפרויקט

כשקונים דומיין (שם אתר) ומעלים שרת לענן, צריך לחבר ביניהם בהגדרות ה-DNS: "הכתובת `myapp.com` תפנה לכתובת ה-IP הזו." תהליך זה לוקח בדרך כלל עד 48 שעות להתפשט ברחבי הרשת.
