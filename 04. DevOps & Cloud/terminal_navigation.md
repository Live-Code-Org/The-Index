# ניווט והתמצאות בטרמינל (Terminal Navigation)

ברוכים הבאים לעולם הטרמינל! כאן אנחנו לומדים איך לדבר עם המחשב ישירות, בלי מתווכחים ובלי ממשקים גרפיים כבדים. זה המקום שבו תרגישו מפתחים אמיתיים בפעם הראשונה.

---

## ⚙️ טרמינל (Terminal / Command Line)
>חלון שיחה ישיר עם מערכת ההפעלה של המחשב, שבו אנו נותנים פקודות בטקסט במקום ללחוץ על כפתורים.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)
* **אפשרות א' (מכשיר הקשר של המפקד):** דמיינו שאתם מפקדים בשטח. במקום לרוץ לכל חייל ולגעת לו בכתף (לחיצה עם העכבר), אתם משתמשים במכשיר קשר (הטרמינל) ופשוט אומרים בקול: "כוח א', לנוע ימינה". מהיר, חד וישיר.
* **אפשרות ב' (הכישוף הסודי של הארי פוטר):** במקום להרים חפצים בידיים (ממשק גרפי), הטרמינל הוא כמו שרביט קסמים. אתם לוחשים לחלון מילה קסומה ספציפית (`Lumos`, או במקרה שלנו `ls`), והמחשב מיד מבצע את הפעולה.

### 💻 קוד תמציתי (Bash)
```bash
echo "Hello, Live Code!" # הדפסת טקסט ישירות על המסך
whoami                  # בדיקה מי המשתמש שמחובר כרגע
```

### 🛠️ יישום בפרויקט
בכל פרויקט שתפתחו, הטרמינל יהיה כלי העבודה המרכזי שלכם – דרכו תריצו שרתים מקומיים (`npm run dev`), תורידו ספריות חיצוניות ותעלו את הקוד ל-GitHub.

## ⚙️ cd (Change Directory)
>שינוי התיקייה הנוכחית ומעבר לתיקייה אחרת במחשב.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)
* **אפשרות א' (מעבר בין חדרים בבית):** הטרמינל הוא כמו הבית שלכם. כרגע אתם עומדים בסלון. כשאתם מריצים `cd kitchen`, אתם פשוט פותחים את הדלת ועוברים פיזית לחדר המטבח.
* **אפשרות ב' (לחיצה כפולה על תיקייה):** בממשק הרגיל של ווינדוס, כדי להיכנס לתיקייה אתם עושים דאבל-קליק עם העכבר. בטרמינל, הפקודה `cd` היא פשוט הדאבל-קליק שלכם במקלדת.

### 💻 קוד תמציתי (Bash)
```bash
cd projects/my-app  # מעבר לתוך תיקיית הפרויקט
cd ..               # חזרה צעד אחד אחורה לתיקיית האב
cd ~                # מעבר מהיר לתיקיית הבית הראשית של המשתמש
```

### 🛠️ יישום בפרויקט
כאשר תרצו להריץ בדיקות או להתקין ספריות בפרויקט מסוים, תצטרכו קודם כל "לנווט" לתוך התיקייה שלו באמצעות `cd`, אחרת המחשב לא ידע על איזה קוד אתם מנסים לעבוד.

## ⚙️ ls (List)
>הצגת רשימה של כל הקבצים והתיקיות שנמצאים בתוך המיקום הנוכחי שלכם.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)
* **אפשרות א' (הדלקת האור בחדר):** נכנסתם לחדר חשוך (או לתיקייה חדשה בטרמינל) ואתם לא רואים כלום. הפעלת `ls` היא כמו להדליק את האור בחדר כדי לראות אילו רהיטים וחפצים מונחים מסביבכם.
* **אפשרות ב' (תפריט במסעדה):** התיישבתם בשולחן. הפקודה `ls` היא המלצר שמגיש לכם את התפריט שמראה לכם בדיוק אילו מנות (קבצים) זמינות עבורכם ברגע זה.

### 💻 קוד תמציתי (Bash)
```bash
ls      # הצגת רשימת הקבצים הרגילה בתיקייה
ls -la  # הצגת כל הקבצים כולל קבצים נסתרים (כמו .git) עם פרטים מלאים
```

### 🛠️ יישום בפרויקט
לפני שמתחילים לעבוד על קובץ הגדרות או קוד, נריץ `ls` כדי לוודא שאנחנו אכן נמצאים בתיקייה הנכונה ושהקובץ שאנחנו מחפשים קיים שם.

## ⚙️ clear
>ניקוי של כל מה שכתוב על המסך בטרמינל והחזרת הסמן לשורה הראשונה.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)
* **אפשרות א' (לוח מחיק בכיתה):** המורה כתב והסביר, הלוח התמלא בנוסחאות ושרטוטים ואין יותר מקום לכתוב. פקודת `clear` היא כמו לקחת ספוג ולהעביר ניגוב מהיר על הלוח כדי שנוכל להתחיל שיעור חדש בראש נקי.
* **אפשרות ב' (סידור השולחן):** השולחן שלכם עמוס בניירות, כוסות קפה ריקות ושאריות דפים. הרצת `clear` היא כמו להעיף את הכל לתוך המגירה ברגע אחד כדי שיהיה לכם מרחב עבודה נקי ומסודר בעיניים.

### 💻 קוד תמציתי (Bash)
```bash
clear  # מנקה את המסך ומחזיר אתכם לדף חלק (אפשר גם להשתמש בקיצור Ctrl+L)
```

### 🛠️ יישום בפרויקט
כשמריצים שרת מקומי שמדפיס המון הודעות לוג או שגיאות, המסך הופך למבולגן ומבלבל מאוד. שימוש ב-`clear` עוזר לנו לאפס את העיניים ולהתמקד רק בשגיאה הבאה שתופיע.
