# חיפוש והרשאות מערכת (Search & Permissions)

בניהול שרתים ותשתיות, לפעמים אנו נדרשים לבצע משימות ממוקדות מאוד – כמו לאתר מילת מפתח ספציפית מתוך מיליוני שורות, או לבצע פעולות רגישות במערכת שרק למנהלי מערכת מותר לבצע.

---

## ⚙️ grep (Global Regular Expression Print)
>חיפוש, סינון והצגה של שורות המכילות מילה או תבנית ספציפית בתוך קבצי טקסט.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)
* **אפשרות א' (פונקציית Ctrl+F של הטרמינל):** כמו שאתם מחפשים מילה ספציפית בדפדפן או במסמך Word ארוך וכל המקומות שבהם היא מופיעה מודגשים בצהוב.
* **אפשרות ב' (מסננת פסטה):** יש לכם סיר ענק עם מים ופסטה (כל הטקסט בקובץ). אתם שופכים את הכל למסננת (grep) – המים עוברים ונעלמים, ורק הפסטה (השורות שרציתם למצוא) נשארת אצלכם ביד.

### 💻 קוד תמציתי (Bash)
```bash
grep "ERROR" server.log        # הצגת כל השורות שמכילות את המילה ERROR בקובץ הלוג
grep -i "database" config.json  # חיפוש ללא רגישות לאותיות גדולות/קטנות (i = ignore case)
```

### 🛠️ יישום בפרויקט
כשמערכת קורסת ויש לנו קובץ לוג באורך קילומטר, נשתמש ב-`grep` כדי לסנן החוצה את כל השורות התקינות ולהציג רק את שורות ה-`Error` או ה-`Exception` כדי להבין מיד מה נשבר.

## ⚙️ sudo (Superuser Do)
>הרצת פקודות בהרשאות של מנהל מערכת (Root) המאפשרות לבצע שינויים רגישים במחשב.

### 🎭 אפשרויות לאנלוגיה (לבחירה ידנית)
* **אפשרות א' (חתימת הורים / אישור מנהל):** דמיינו שאתם תלמידים בבית ספר שרוצים לצאת לטיול שנתי. המאבטח בשער (מערכת ההפעלה) לא ייתן לכם לצאת סתם כך, אלא אם תציגו לו אישור חתום על ידי ההורים (הפקודה `sudo`).
* **אפשרות ב' (מפתח המאסטר של הבניין):** אתם יכולים להיכנס לדירה שלכם בלי בעיה. אבל אם אתם צריכים להיכנס לחדר הגנרטורים או ללוח החשמל הראשי של הבניין – אתם צריכים לבקש מאב הבית את מפתח המאסטר שלו.

### 💻 קוד תמציתי (Bash)
```bash
sudo apt update          # עדכון רשימת החבילות במערכת (דורש הרשאות מנהל)
sudo rm -rf /var/log/tmp # מחיקת קבצי לוג זמניים של המערכת
```

### 🛠️ יישום בפרויקט
כאשר נרצה להתקין תוכנות גלובליות על השרת שלנו (כמו Nginx או Docker), המערכת תחסום אותנו כברירת מחדל כדי להגן על עצמה. שימוש ב-`sudo` יבהיר למערכת שאנחנו יודעים מה אנחנו עושים ויאפשר את ההתקנה.
